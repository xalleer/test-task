## Install and run app

1. Clone this repository
2. Write "npm install" to console
3. After install all packages write "npm run dev" to console
4. Go to "localhost:5173"

**** 

## Build app

1. Clone this repository
2. Write "npm install" to console
3. After install all packages write "npm run build" to console
